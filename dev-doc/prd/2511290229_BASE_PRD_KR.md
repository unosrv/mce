# 업무 내용

## 프로젝트 개요

- 축산농가(한국/미국/브라질)의 온실가스 배출량을 측정하고 탄소배출 리포트를 자동 생성하는 SaaS 플랫폼 구축
- 농가가 가축 정보, 사료, 사육환경, 분뇨관리 데이터를 입력하면 IPCC Tier 2 방법론으로 배출량을 계산하여 리포트를 제공함

## 필요 요소

- 웹 프런트엔드 개발
- 백엔드 개발
- 관리자페이지 개발
- 서버/DB/인프라 구성

## 지원 플랫폼

- PC Web (모바일 대응 X)

## 개발 환경/언어

- 랭기지: Node.js 또는 Python (그 외 랭기지 사용 X)
- DBMS: PostgreSQL 또는 MySQL
- 그 외 스택 제안해주세요

## 의뢰자가 제공하는 사항

- 상세 작업의뢰서 (A4 3p)
- ERD
- Figma 디자인 제공 예정

## 주요 기능

### 사용자 인증
- 이메일 기반 로그인
- 닉네임 사용

### 농장 관리
- 다중 농장 등록
- 국가별 주소 입력 (Google Maps)
- 회계연도 설정

### 가축 그룹 관리
5단계 입력 폼:

1. **기본정보**: 종류/두수/체중/유량
2. **사료**: 최대 10개, 프리셋 지원, 첨가제
3. **사육환경**: 월별 축사/운동장/방목 비율
4. **축사정보**: 면적/온도/유형
5. **분뇨관리**: 분기 로직 (혐기성 소화조/고형액상 분리기/처리 시스템)

### 배출량 리포트
- 농장 단위 자동 생성
- 카테고리별 배출량 (장내발효/분뇨/사료/에너지)
- 가축 그룹별 상세
- PDF 내보내기

### 대시보드
- 전체 농장 통계
- 월별 추이
- 농장별 비교

## 필수 자격 요건

- 3주 단기 개발 가능한 팀
- 복잡한 비즈니스 로직 구현 경험 필수
- 데이터베이스 설계 역량 (22개 테이블, 복잡한 관계) 보유
- 다국어 서비스 개발 경험 (영어/한글, 자동 단위 변환 등)
