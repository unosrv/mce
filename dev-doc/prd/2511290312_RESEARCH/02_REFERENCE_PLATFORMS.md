# Reference Platforms for UI/UX Research

## Overview

This document catalogues farm carbon footprint calculator platforms that serve as references for UI/UX design. The client-provided references (Cool Farm Tool, COMET-Farm) have functional but outdated designs. Better UI/UX inspiration is needed to win the project bid.

---

## Tier 1: Primary References (Client Provided)

### 1. Cool Farm Tool
- **URL**: https://app.coolfarmtool.org/
- **Provider**: Cool Farm Alliance
- **Users**: 150+ countries, tens of thousands of users
- **Methodology**: IPCC-based, peer-reviewed
- **Features**:
  - GHG emissions calculation
  - Biodiversity metrics
  - Water use measurement
  - Food loss/waste tracking
- **UI Assessment**: Functional but dated; form-heavy interface
- **Strengths**: Comprehensive methodology, global adoption

### 2. COMET-Farm
- **URL**: https://comet-farm.com/
- **Provider**: Colorado State University / USDA
- **Focus**: US agricultural GHG accounting
- **UI Assessment**: Academic/government style; complex navigation
- **Strengths**: Robust calculation engine, USDA backing

## UI/UX Design Recommendations

### Based on Research Findings

#### 1. Form Design
- **Progressive disclosure**: Multi-step forms with clear progress
- **Contextual help**: Tooltips for technical terms
- **Auto-save**: Prevent data loss
- **Smart defaults**: Pre-populate common values

#### 2. Dashboard Design
- **KPI cards**: At-a-glance metrics
- **Interactive charts**: Drill-down capability
- **Comparison views**: Farm-to-farm, period-to-period
- **Export options**: PDF, CSV, share links

#### 3. Visual Language
- **Colour scheme**: Green/earth tones (environmental context)
- **Icons**: Simple, recognisable agricultural symbols
- **Typography**: Clean, readable sans-serif
- **Spacing**: Generous whitespace for clarity

#### 4. Mobile Considerations
- **Note**: PRD specifies PC Web only
- **Recommendation**: Use responsive breakpoints anyway for future-proofing
